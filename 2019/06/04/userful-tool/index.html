我在使用的一些工具

Shino161
2019-06-04
工具

文档工具

Dash

  • 非常好用的查询工具，200元用一辈子，爽到。

    Mac的备忘录

  • 经常用来记一些配置或者开发中常用的命令，很便利

    OneNote

  • 在win上的一款工具，和mac的备忘录类似。但是没有很深度的使用。

    开发工具

    Atom

  • 刚入行的时候用的编辑器,还算顺手，但经常会出现内存占用过高的问题，遗憾。

    VS Code

  • 这个就8⃣️用说了8⃣️。强大的扩展程序，稳定性和易用性上都不错。比较遗憾是想在分屏上看代码要再开一个VS Code。

    Sequel Pro

  • 关系型数据库的GUI工具。从入行到现在都一直在用的工具。顺手好用，推荐。

    Sourtree

  • Git的GUI工具，还算是不错。但是远端的提交量大的时候，会一直loading。这个比较蛋疼。

    PerfectPixel

  • Chrome浏览器的一个插件，可以将图片作为一个半透明的层覆盖在网页上。Web前端做视觉实现的无敌插件，推荐。

    whistle

  • 一个跨平台抓包调试工具，很好用。对于开发app或者微信H5等帮助很大。
