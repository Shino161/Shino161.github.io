Antd的Input组件自动填充问题

环境

  • chrome 80+

  • Antd 4+

解决思路

其实解决的方法是参照 MDN 即可。

https://developer.mozilla.org/zh-CN/docs/Web/Security/Securing_your_site/Turning_off_form_autocompletion

比较神奇的地方就是，在尝试在 Form 组件和 name=’username’ 的 Input 组件上尝试 autoComplete=”off” 和 autocomplete=”nope” 均是无效的。

当在name=’password’ 的 Input 组件上使用 autoComplete=’new-password’ 才阻止了chrome 的自动填充。
